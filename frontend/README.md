# Frontend - Ứng dụng Tin tức

Frontend React + Tailwind CSS cho ứng dụng tin tức.

## Cài đặt
1. `npm install`
2. `npm run dev` để khởi động server phát triển trên http://localhost:5173

## Build
`npm run build`

## Tính năng
- Trang chủ: Danh sách tin tức
- Đăng nhập
- Chi tiết tin tức với bản đồ và bình luận
- Bảng điều khiển Admin: Thống kê, quản lý người dùng
- Bảng điều khiển Editor: Thống kê cá nhân, quản lý tin tức
